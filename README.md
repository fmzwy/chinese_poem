# 拼拼古诗

一个古诗评图游戏

包含唐诗三百首，和小学、初中、高中课本中的古诗。

- 可以切换简体繁体。

- 每个字可以展示两种风格的汉语拼音

- 可以展示每句古诗的英文翻译

- 可以随机显示一些字作为提示
